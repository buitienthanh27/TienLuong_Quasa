## 1. Tổng Quan Hệ Thống

### 1.1 Mục đích
Hệ thống quản lý toàn diện về nhân sự, chấm công, tính lương và phân bổ chi phí cho doanh nghiệp nông nghiệp/chế biến.

### 1.2 Người dùng mục tiêu
| Role | Mô tả | Quyền hạn |
|------|-------|-----------|
| Admin | Quản trị viên hệ thống | Toàn quyền |
| HR Manager | Quản lý nhân sự | Quản lý nhân viên, chấm công |
| Accountant | Kế toán | Tính lương, phân bổ chi phí |
| Viewer | Xem báo cáo | Chỉ xem |

## 8. Ghi Chú Cho Designer

1. **Figma Organization**: Tạo các page riêng cho mỗi module (Dashboard, Danh mục, Nghiệp vụ, Báo cáo, Cài đặt)

2. **Component Library**: Xây dựng Design System trước với:
   - Colors, Typography
   - Buttons, Inputs, Selects
   - Tables, Cards, Modals
   - Icons (recommend: Material Icons hoặc Heroicons)

3. **Interactions**: Document các trạng thái:
   - Default, Hover, Active, Disabled
   - Loading states
   - Empty states
   - Error states

4. **Accessibility**: Đảm bảo contrast ratio WCAG AA (4.5:1 cho text)

5. **Vietnamese Text**: Sử dụng tiếng Việt có dấu đầy đủ
