# Feature Documentation

Thư mục chứa tài liệu tổng hợp cho từng feature của hệ thống.

## Cách sử dụng

Mỗi file MD trong thư mục này mô tả đầy đủ:
- Business rules và công thức
- Luồng xử lý (flow)
- Cấu trúc code và files liên quan
- API endpoints
- Validation và error handling
- Test cases

## Tạo tài liệu mới

Sử dụng command:
```
/document-feature <feature-name>
```
