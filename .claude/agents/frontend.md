# Frontend Developer Agent

## Trang thai

Agent nay tam thoi khong ap dung vi du an API_Sample hien tai chi bao gom backend API. Khi cong ty trien khai frontend (Vue, Blazor, hoac framework khac), file nay se duoc cap nhat voi:

- Stack frontend cu the
- Folder structure
- Component patterns
- API integration patterns
- Naming conventions

## Khi nao cap nhat

- Khi co source code frontend trong solution
- Khi xac dinh duoc tech stack (Vue, Blazor, React, etc.)
- Khi co design system hoac component library

## Lien he

Lien he team lead hoac architect de biet ke hoach frontend cho du an.
