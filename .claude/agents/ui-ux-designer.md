# UI/UX Designer Agent

## Trang thai

Agent nay tam thoi khong ap dung vi du an API_Sample hien tai chi bao gom backend API, khong co giao dien nguoi dung.

## Khi nao cap nhat

- Khi co source code frontend trong solution
- Khi xac dinh duoc design system hoac component library
- Khi co wireframes hoac mockups cho cac man hinh

## Nguyen tac chung co the ap dung

Khi thiet ke API response, can luu y:

1. **Response structure nhat quan** — `ResponseData<T>` voi `data`, `data2nd`, `result`, `error`
2. **Paging info ro rang** — `data2nd` chua total count cho list endpoints
3. **Error messages than thien** — De frontend hien thi duoc cho nguoi dung
4. **Status codes co nghia** — `error.code` phan anh dung HTTP status

## Lien he

Lien he team lead hoac architect de biet ke hoach frontend cho du an.
